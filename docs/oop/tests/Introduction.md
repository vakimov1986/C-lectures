# Вступ

## Масиви

У C# структури даних, що можуть містити нуль або більше елементів, називаються колекціями. **Масив — це колекція фіксованого розміру**, всі елементи якої мають бути одного типу. Доступ до елементів масиву здійснюється за індексом. Масиви в C# мають нумерацію з нуля, тобто перший елемент має індекс `0`:

```csharp
// Оголошення масиву з явним розміром (розмір — 2)
int[] twoInts = new int[2];

// Присвоєння другого елемента за індексом
twoInts[1] = 8;

// Отримання другого елемента за індексом
twoInts[1] == 8; // => true
```

Масив можна створити також скороченим способом, одразу задаючи значення. Оскільки компілятор сам визначає кількість елементів, розмір можна не вказувати:

```csharp
// Три еквівалентні способи оголосити й ініціалізувати масив (розмір — 3)
int[] threeIntsV1 = new int[] { 4, 9, 7 };
int[] threeIntsV2 = new[] { 4, 9, 7 };
int[] threeIntsV3 = { 4, 9, 7 };
```

Маніпулювати масивами можна як через їхні методи й властивості, так і через статичні методи класу `Array`.

## Цикл `for`

Оператор `for` дозволяє багаторазово виконувати код у циклі, доки умова вірна.

```csharp
for (int i = 0; i < 5; i++)
{
    System.Console.Write(i);
}

// => 01234
```

Цикл `for` складається з чотирьох частин:

1. **Ініціалізатор** — виконується один раз перед стартом циклу і створює початкові змінні.
2. **Умова** — перевіряється перед кожною ітерацією. Цикл виконується, доки умова істинна.
3. **Ітератор** — виконується після кожної ітерації (найчастіше інкремент або декремент).
4. **Тіло** — код, що виконується на кожній ітерації.

## Цикл `foreach`

Оскільки масив — це колекція, окрім доступу за індексом можна перебирати його значення за допомогою `foreach`:

```csharp
char[] vowels = new [] { 'a', 'e', 'i', 'o', 'u' };

foreach (char vowel in vowels)
{
    System.Console.Write(vowel);
}

// => aeiou
```

# Завдання

Ти — завзятий орнітолог, який веде облік птахів, що прилітали до твого саду за останні сім днів.

Є шість завдань, кожне з яких пов’язане з кількістю птахів.

---

### **1️⃣ Метод `LastWeek()`**

Для порівняння ти завжди тримаєш запис минулого тижня:

```
0, 2, 5, 3, 7, 8, 4
```

Реалізуй статичний метод:

```csharp
BirdCount.LastWeek();
// => [0, 2, 5, 3, 7, 8, 4]
```

---

### **2️⃣ Метод `Today()`**

Повертає кількість птахів за поточний день (останній елемент):

```csharp
int[] birdsPerDay = { 2, 5, 0, 7, 4, 1 };
var birdCount = new BirdCount(birdsPerDay);
birdCount.Today();
// => 1
```

---

### **3️⃣ Метод `IncrementTodaysCount()`**

Збільшує значення останнього дня:

```csharp
birdCount.IncrementTodaysCount();
birdCount.Today();
// => 2
```

---

### **4️⃣ Метод `HasDayWithoutBirds()`**

Повертає `true`, якщо у якийсь день було `0` птахів:

```csharp
birdCount.HasDayWithoutBirds();
// => true
```

---

### **5️⃣ Метод `CountForFirstDays(int days)`**

Повертає суму птахів за перші `days` днів:

```csharp
birdCount.CountForFirstDays(4);
// => 14
```

---

### **6️⃣ Метод `BusyDays()`**

"Зайнятий" день — коли ≥5 птахів:

```csharp
birdCount.BusyDays();
// => 2
```
