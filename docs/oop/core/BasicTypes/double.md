# Тип `double`

Тип `double` використовується для зберігання чисел з плаваючою комою (десяткових значень). Він підтримує як додатні, так і від’ємні значення та застосовується тоді, коли потрібна робота з дробовими числами.

У C# `double` — це **64-бітне число з плаваючою комою**, тобто для зберігання значення використовується **64 біти (8 байтів)** пам’яті.

---

## Оголошення змінних типу `double`

```csharp
double temperature = 36.5;
double pi = 3.14159;
double negativeValue = -273.15;
```

---

## `double` vs `int`

Цілі числа (`int`) можуть зберігати лише значення без дробової частини. Якщо потрібна точність для дробових значень, слід використовувати `double`.

```csharp
int wholeNumber = 36;         // Дробова частина недопустима
double preciseValue = 36.5;   // Дробові значення зберігаються
```

---

## Використання десяткової крапки

Під час присвоєння значень типу `double` рекомендується явно використовувати десяткову крапку.

```csharp
double exactValue = 100.0;   // Явно вказано тип double
double fraction = 0.5;       // Дробове значення
double negative = -40.0;     // Від’ємне дробове число
```

## Діапазон значень double 

double — це 64-бітне число з плаваючою комою, яке дозволяє зберігати дуже великі, дуже малі та спеціальні значення.

| Властивість              | Значення                   |
| ------------------------ | -------------------------- |
| Розмір                   | 64 біти (8 байтів)         |
| Максимальне значення     | ≈ 1.79 × 10<sup>308</sup>  |
| Мінімальне від’ємне      | ≈ -1.79 × 10<sup>308</sup> |
| `NaN`                    | Not a Number (не число)    |
| `PositiveInfinity`       | +∞                         |
| `NegativeInfinity`       | −∞                         |

Ці значення можна отримати безпосередньо в коді:

```csharp
double min = double.MinValue;
double max = double.MaxValue;

//невизначена операція : можна отримати при діленні 0.0 / 0.0, ...
double nan = double.NaN; 
// Ділення додатного числа на 0, Переповнення додатнього значення, ...
double posInf = double.PositiveInfinity; 
// Ділення від’ємного числа на 0, Переповнення від’ємного значення
double negInf = double.NegativeInfinity; 
```
