# Тип `int`

Тип `int` використовується для зберігання **цілих чисел** (без десяткової частини). Він підтримує як **додатні**, так і **від’ємні** значення.
У C# `int` — це **32-бітне знакове ціле число**, тобто для зберігання значення використовується **32 біти (4 байти)** пам’яті.

---

## Оголошення змінних типу `int`

### Оголошення та ініціалізація в одному рядку

```csharp
int age = 25;
int year = 2024;
```

### Оголошення спочатку, присвоєння пізніше

```csharp
int score;
score = 100;
```

---

## Додатні та від’ємні числа

```csharp
int temperature = -10;   // від’ємне число
int altitude = 8848;     // додатне число
int balance = 0;         // нуль — також допустиме значення
```

---

## Базові математичні операції з `int`

```csharp
int a = 10;
int b = 3;

int sum = a + b;          // 13
int difference = a - b;   // 7
int product = a * b;      // 30
int division = a / b;     // 3
```

⚠️ Зауваження: при діленні двох `int` результат також буде `int` (дробова частина відкидається).

---

## Діапазон значень `int` (32-бітний)

Оскільки `int` є 32-бітним **знаковим** цілим числом, він може зберігати значення в такому діапазоні:

| Властивість          | Значення                            |
| -------------------- | ------------------------------------|
| Розмір               | 32 біти (4 байти)                   |
| Мінімальне значення  | -2<sup>31</sup> = -2 147 483 648    |
| Максимальне значення | 2<sup>31</sup> - 1 = 2 147 483 647  |

Ці межі можна отримати безпосередньо в коді:

```csharp
int min = int.MinValue;
int max = int.MaxValue;
```

---

## Коли використовувати `int`

Тип `int` підходить для:

* віку, років, кількості
* лічильників і циклів
* індексів масивів і колекцій
* будь-яких цілих значень у стандартному діапазоні
